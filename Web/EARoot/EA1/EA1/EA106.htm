If Solution1 (solution/advice based on old knowledge) is the same as Solution2 (solution/advice based on new knowledge) then there is no problem!  <br/></p><p>b. If Solution1 is different from Solution2, then there might be a problem:<br/></p><p>1. Check if it’s an issue that the solutions are different.<br/></p><p>2. Figure out how to resolve the problem ? might mean that Knowledgebase needs changes.<br/></p>
